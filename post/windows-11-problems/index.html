Windows 11 存在的一些问题
发布于 2022-01-21，包含标签 。

本人用 Windows 11 已经用几个月了。虽然刚用的时候确实被其新貌给惊艳到了，但用久了就发现了其背后的问题。本人便在此文章中列出 Windows 11 中的五个明显的且所有设备都存在的问题，同诸位分享。（所以在这五个问题中，没有一个问题是和系统稳定性有关的，因为这个问题很多时候因设备而异。）

注意：这里的大部分观点都是个人观点！

1、一个顶栏，两边不一。

Windows 11 对大部分系统内置应用的风格进行了更新。它们乍一看挺好看的，但在部分应用（详细点，指 Microsoft Store 和系统设置）仔细观察就会发现，顶栏的两边是没有对齐的。左边的文字或箭头或图标，相对于右边的三个按钮偏下了。（不过这样的问题也有一定的原因在里面）

2、某些字体，上古世纪。

Windows 11 对固定在任务栏上的应用的预览栏风格进行了更新。但更新的同时却引来了“倒退”。
预览栏上显示的文字由系统早期推送版本时的“微软雅黑”变成了之后的“宋体”。这让用户看起来违和感满满，还降低了系统在外貌上的逼格。（估计强迫症患者注意到这个病就来了）

3、好看的动效，并非“人人”都有。

Windows 11 更新了应用最小大化的动效，大部分应用都已经适配了。
但也仅仅是大部分，还有应用存在没有适配的情况（注：可能不关系统的事，与应用是否愿意主动适配有关）。
（具体看本人在 B 站发布的同题视频）

4、风格变了，但没全变。

Windows 11 带来了全新的 UI 风格，同时也还留着 Windows 10 的老问题——设计风格不统一。
在 Win11 上，大部分的系统内置应用都已经换了新，但还有个别应用（详细点，指任务管理器、邮箱和日历、Windows 安全中心等应用）还留在旧版本的风格。


而且，在某些用上新风格的应用上，还能扒出些熟悉的东西。如此多老旧的痕迹混杂在里面，且又是从 Win10 遗留下来的诟病，这就让系统在外观上的逼格降了不少，也影响到用户的视觉感受。（除非用户不在意）

不过据消息报告，在最新的 Dev 版本上，任务管理器和 Windows Defender 都已经换新，这意味着更多的应用也都能换上新的风格，系统的 UI 的风格统一更近了一步（不过风格统一的完成就是时间问题了）。

5、某些好用的功能，被大刀部砍去了。

Windows 11 增添了很多新奇的功能，但也砍掉了一些在旧版本中用起来非常好用的功能。比如磁贴和任务栏应用拖放固定这两个功能。
磁贴最早被应用于 Windows Phone，后来被应用于 Windows 8，并被传承到了 Windows 10 上。因信息能在磁贴上预览和利用磁贴可以对开始菜单或屏幕进行个性化定制等功能，磁贴深受广大用户的好评。但不知何原因，在 Windows 11 上，磁贴被改成了纯应用菜单，这也就意味着开始菜单不再变得像之前那样好玩，也意味着 Windows 11 的个性化程度又少了积分。（说白了，就是让非常喜欢磁铁的用户对其说一声：RNM，退钱！）

在 11 之前的 Windows 版本上，要在任务栏上固定应用，只需要直接将应用给拖进任务栏上，或者右键菜单选择“固定到任务栏上”即可，而将应用直接拖到任务栏上，本人相信这比右键菜单操作会方便很多，也相信很多用户都是这样将应用固定在任务栏上的。但可惜的是，在 Windows 11 上，这个功能被大刀部给砍了。（RNM，退钱！）

以上，就是本人和诸位所分享的，Windows 11 中存在的五个明显问题。除了这几个明显问题，Win11 还有着诸多影响生产力的、与系统稳定性相关的问题，由于这些问题很多时候因设备而异，且篇幅有限，本人便不一一说出来了。纵观本人上述的问题，仔细想下可以发现，这些明显的问题和两个因素有关：对细节的考虑和所处的角度。由于微软对细节方面的考虑不足，就造成了上述问题中的前四个（第三个除外），而剩余的问题，就和微软没有站在用户的角度有关了。想要让 Windows 11 成为其宣传语所说的“更接近于用户所爱”的模样，如果微软未能及时发现这些问题，并考虑造成这些问题的因素，恐怕还需要一段时间。或者说，现在，还为时过早了。
