Gemini UI 1.0 Offcial Edition
发布于 2021-09-12，包含标签 # Gemini UI。

主要介绍

Gemini UI 是由 Lipus Studio 制作的、基于 Microsoft PowerPoint 的仿操作系统 UI 。UI 使用以 Level、Simple、Material、Smooth 为设计核心的 G-Level Design 设计语言（具体在之后会较为详细地说明），界面好看、简洁，动画以平滑和流畅为主。
“少搞花里胡哨的东西、削减华而不实的功能、让系统中的各个应用都有其用。”在 PPT 的诸多限制下，UI 中大部分应用本该有的功能都尽量地被还原（虽然最终只是玩玩而已），在日常中很少用上或者根本用不上的功能在此 UI 中都被砍掉，这是本 UI 的特点，也是工作室对制作 PPTUI 的要求之一。
